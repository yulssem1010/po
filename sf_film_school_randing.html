지인으로부터 SF FILM SCHOOL이라는 그래픽 전문 학원을 추천 받아 해당 사이트에 방문하였는데 글의 크기가 작아 한 눈에 보기 어렵고, 이미지와 글이 너무 붙어 있어 눈에 잘 들어오지 않는다.  또한 원하는 정보를 얻으려면 메뉴 카테고리에서 해당 내용을 찾아 들어가야 하지만 카테고리의 구성도 복잡하고 일일히 찾아 들어가는데에 번거로움을 느껴 정보를 얻는데에 어려움을 겪게 되는데,  이 모든 문제를 해결할 순 없을까?

Pain-point
  • 메인 페이지의 글자가 너무 작아 보기가 불편함
  • 일일히 카테고리를 찾아 들어가기 번거로움
needs
  • 필요한 정보를 한 눈에 보기를 원함
  • 복잡하지 않고 정확한 카테고리 구성

problem

기존 웹 사이트의 문제점을 파악하였습니다.

  • problem 01

    많은 양의 이미지와 글

    다양하지만 내용이 중복되는 사진과 글 등의 정보가 잡다하게 나열되어 있어 직관성과 가독성이 떨어짐

  • problem 02

    사이즈가 작은 글자와 이미지

    글자와 이미지의 크기까지 작아 사용자가 보기에도 클릭하기에도 불편함.

  • problem 03

    컨텐츠 사이의 간격과 정렬

    모든 글이 왼쪽으로 정렬되어 있어 심미성이 떨어짐.
    컨텐츠 간의 간격이 부족하여 사용자가 한 눈에 정보를 얻기가 어려움.

기존의 복잡하고 중복되어 있던 네비게이션을 알맞는 카테고리 별로 묶어 기획하였으며, 글자 크기를 키워 사용자가 마우스로 클릭하는데에 불편함이 없도록 디자인하였다.

포트폴리오

기존 수강생이 작업한 작품의 영상을 보여준다.
사용자가 원하는 부분을 다시 볼 수 있고, 전체 화면으로 크게 볼 수 있도록 컨트롤 바를 추가하였다.

커리큘럼

과정별 대표 포트폴리오 썸네일 이미지를 배치하고, 마우스를 갖다 댔을 때 이미지가 커지며 그에 해당하는 정규 과정 이름이 나타나도록 디자인하였다.

수강생 인터뷰

기존 수강생의 후기를 궁금해하는 사용자들을 위해 수강 후기를 간략하게 표기하였고, 마우스를 갖다 댔을 때 통통 튕기는 듯한 효과가 나타나도록 디자인하였다.

SF FILM SCHOOL 만의 다양한 활동

학원에서 주최하는 대표적인 활동과 그에 대한 자세한 내용을 기재하여 SF FILM SCHOOL만의 차별점을 나타낼 수 있도록 기획하였고, 아코디언 디자인을 이용하여 시각적인 복잡성을 줄이고 콘텐츠에 집중할 수 있도록 디자인하였습니다.

취업 연계 업체

본원에서 취업과 관련하여 연계해주는 업체들을 자동 슬라이드 방식으로 나열하였으며, 업체 로고 클릭 시 해당 업체의 사이트로 이동할 수 있다.
